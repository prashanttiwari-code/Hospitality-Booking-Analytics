**Project Overview**
This project focuses on analyzing hotel booking data using Microsoft Excel to generate meaningful business insights.
The analysis includes data cleaning, data processing, pivot table creation, and an interactive dashboard. 
The goal is to transform raw booking data into actionable insights that help stakeholders make informed, data-driven decisions in the hospitality domain.

**Data Source**
The dataset used in this project was sourced from Kaggle, a popular platform for open datasets and data science projects. 
The data was further cleaned and processed using Microsoft Excel for analysis and visualization.

**Objectives**
1.Clean and preprocess raw hotel booking data
2.Analyze booking patterns and customer behavior
3.Identify trends in bookings and revenue
4.Create an interactive dashboard for easy data visualization

**Tools & Technologies**
1.Microsoft Excel
2.Data Cleaning
3.Data Processing
4.Pivot Tables
5.Charts & Visualizations
6.Interactive Dashboard
